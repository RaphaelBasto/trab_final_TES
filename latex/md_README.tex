Depois do projeto clonado e do SQL inserido no banco, você precisará entrar na pasta do projeto e alterar o arquivo \mbox{\hyperlink{conexao_8php}{conexao.\+php}} para as suas variáveis do MYSQL (user, dbname, password, hostname).

Depois destas etapas feitas, o projeto já estará rodando.
